Dice Instance Table

A dice instance inherits all of its methods from the dice class. 

Dice_Instance = {
    num = number,
    faces = number,
    sets = number,               -- optional
    bonus = number,              -- optional
    is_bonus_plural = boolean,   -- optional (requires bonus)
    rerolls = number,            -- optional
    is_reroll_plural = boolean,  -- optional (requires rerolls)
    minimum = number/nil,        -- optional (nil results in class default (if any) )
